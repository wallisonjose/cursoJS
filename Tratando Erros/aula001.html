Tratando erros

Os principais comandos para tratativa de erros são:

  • try: Que irá testar um bloco de código, ele deve ser usado em conjunto com o próximo comando
  • catch(error): Ele cria uma espécie de variável cujo nome fica entre parentesis que irá receber o erro caso ocorra dentro do bloco try. Além disso é possível que escolhamos a mensagem a ser exibida caso ocorra um erro a inserindo dentro do bloco catch
  • throw: Ela cria um erro que será exibido de acordo com a situação desejada
